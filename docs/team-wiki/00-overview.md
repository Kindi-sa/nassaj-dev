# صورة نسّاج الكبيرة

أهلاً بك. نسّاج منظومة عمل تُساعد الفريق على إنجاز المشاريع بكفاءة. دعنا نفهم الصورة الكبيرة قبل الدخول للتفاصيل.

## الهيكل الكامل

منظومة نسّاج تعيش داخل الكِندِي كوحدة داخلية تخدم الشركة والذراعين:

```mermaid
graph TD
    Kindy["<b>الكِندِي</b><br/>الشركة الأم<br/>الهوية والسياسات والقرارات"]
    
    DarAlKindy["<b>دار الكِندِي</b><br/>الخدمات الاستشارية<br/>والتطوير المخصص"]
    
    OSS["<b>مشاريع OSS</b><br/>المصادر المفتوحة<br/>AlKindy-OSS"]
    
    Nassaj["<b>نسّاج</b><br/>أداة داخلية<br/>لا تُباع"]
    
    NassajCore["<b>نسّاج كور</b><br/>العقل: المنسّق والوكلاء"]
    
    NassajDev["<b>نسّاج ديف</b><br/>الواجهة: الموقع"]
    
    Kindy -->|توجيه استراتيجي| DarAlKindy
    Kindy -->|توجيه استراتيجي| OSS
    Kindy -->|توجيه عملي| Nassaj
    
    Nassaj -.->|خدمة داخلية| DarAlKindy
    Nassaj -.->|خدمة داخلية| OSS
    
    Nassaj --> NassajCore
    NassajCore --> NassajDev
```

### ١. الكِندِي — المظلة

الشركة الأم وإطارها الاستراتيجي. تحتها ذراعان:

- **دار الكِندِي:** الخدمات الاستشارية والتطوير المخصص للعملاء
- **مشاريع OSS:** المصادر المفتوحة (github.com/AlKindy-OSS)

### ٢. نسّاج — الوحدة الداخلية

أداة ذكاء اصطناعي داخلية **لا تُباع ولا تُسوّق**. تخدم الفريق الداخلي فقط. مثل مطبخ المطعم — الزبائن لا يرونها.

**نسّاج جزآن:**

- **نسّاج كور** — العقل: المنسّق والوكلاء والقواعد التنظيمية
- **نسّاج ديف** — الواجهة: الموقع الذي تفتحه بالمتصفح

## رحلة سريعة: ماذا يحدث عندما تكتب طلباً؟

```mermaid
sequenceDiagram
    participant انت as أنت
    participant الموقع as نسّاج ديف
    participant المنسّق as المنسّق
    participant وكيل as وكيل متخصص
    
    انت ->> الموقع: أكتب طلباً
    الموقع ->> المنسّق: يمرر الطلب
    المنسّق ->> المنسّق: يفحص الحصة والموارد
    المنسّق ->> وكيل: يفوّض الوكيل المختص
    وكيل ->> وكيل: ينفّذ ويختبر
    وكيل ->> المنسّق: انتهيت من المهمة
    المنسّق ->> الموقع: يحدّث لوحة المشروع
    الموقع ->> انت: ترى النتيجة مباشرة
```

## المصطلحات الأساسية الأربع

| المصطلح | معناه |
|---|---|
| **المشروع** | مجلد عمل (مثل nassaj-dev): فيه الملفات والكود والمهام |
| **الجلسة** | محادثة مستمرة واحدة مع وكيل حول مهمة محددة |
| **لوحة المشروع** | شاشة تلخص حالة المشروع: المراحل والمهام والأخطاء |
| **الوكيل** | متخصص واحد من فريق الأذكياء: مبرمج، مصمم، محلل، إلخ |
